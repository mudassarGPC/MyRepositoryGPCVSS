# My Portfolio

A professional portfolio website for Mudassar Khattab - Supply Chain and Commerce Technology Leader.

## About

This portfolio showcases my ~20 years of experience across product management, enterprise sales/presales, solution delivery, and data-driven decisioning in supply chain, digital commerce, and omni-channel order management systems (OMS).

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with responsive design
- **Vanilla JavaScript** - No frameworks, no dependencies
- **Supabase** - Backend for email collection

## Features

- Clean, minimal, modern design
- Fully responsive (mobile and desktop)
- Email gate for visitor engagement
- Professional presentation of experience and skills
